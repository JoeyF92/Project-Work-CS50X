About

Hey there I'm Joe and I'm recently moved out to Morocco to take on coding full time. I'm currently working my way through Cs50- and for one of the projects I've been instructed to create a homepage. I thouoght I'd use this opportunity to document parts of my stay

The Rules are as follows:

  • It must have at least 4 different HTML Pages, you have to be able to reach all pages from any page.
  • It must have 10 distinct tags (excluding html, head body and title)
  • It must have one or more bootstrap features (i choose to use a bootstrap navbar)
  • Have one stylesheet of my own creation, use at least 5 diffferent selectors + 5 different properties
  • Integrate one Javascript feature to make site more interactive (click on my name on the homepage)
  • Must look good on mobile and desktop

Things learnt in the project:

  • How to use a video as the background for the page
  • Creating a photo carousel
  • Some nice stylings to make the page seem more interactive - button stylings, key frames
  • Getting better with flexbox and alround mobile design
